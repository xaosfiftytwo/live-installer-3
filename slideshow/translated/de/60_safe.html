Sichere Updates

  • Einige Betriebssysteme machen die Aktualisierung Ihres Systems schwieriger als es wirklich sein muss. Der SolydXK Philosophie zufolge sollten Upgrades unkompliziert, schnell und einfach sein.
  • Der Update Manager wartet im Systemtray and benachrichtigt Sie über die Verfügbarkeit von Updates. Updates werden nur auf Ihren Wunsch hin installiert. Der Update Manager wird kein gesondertes Fenster öffnen oder Sie sonst belästigen.
  • Alle aktualisierten Dateien werden von von vertrauenswürdigen Entwicklern und Maintainern überprüft und gewartet.
  • Alle Dateien werden in speziellen sicheren Repositorys für Ihre Verwendung gespeichert.
  • Das Sicherheitsrepository wird täglich synchronisiert.
